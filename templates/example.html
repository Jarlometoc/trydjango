FibSim example


Below is an example of a typical FibSim output; in this case the fibril 1QL1, a filamentous bacteriophage. Layer-lines data from an X-ray diffraction experiment was also included, therefore a chi-square statistic comparing the model to the experimental data is also presented. Note how one of the advanced settings (Bfactor) has been altered from its default and therefore noted in the output. This run has been named 'The Test' and can be recalled for viewing/downloading by inputing the associated run number in the 'Choose an older run' button.
