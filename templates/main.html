Fibrillar Assembly Server

This is the MainPage for running the analysis. Provide a PDB file, set the parameters and run the program. In return you receive a layline plot, Jmol plot and a fibrillar score. The program takes a few minutes to run. Users can receive results via Email. See the example run on the HomePage, watch the video, or contact us if you have any questions.

Getting ready

Directions

  • RETRIEVE a PDB, either locally or get it from the RCSB Protein Data Bank.
  • Select PARAMETERS, RUN and wait. This will take a few minutes.
  • Examine the results.
  • Adjust parameters and RUN the query again.
  • Press SAVE to send your results to your Email address.
  • Press CLEAR to start over with a new file.
  • Modify the run using the JMOL SLIDER and RERUN
