FibSim example


Below is an example of a typical FibSim output; in this case the displayed data is the fibril 1QL1, a filamentous bacteriophage. Layer-lines data from an X-ray diffraction experiment is also included. Some of the numerous generic and advanced settings have been altered from their defaults. Note how no advanced settings are displayed if they are not altered- you see only those that you change. This run has been named 'The Test' and can be recalled for viewing/downloading by inputing the run number in the 'Choose an older run' button."
